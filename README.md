# 💱 Conversor de Moedas

Aplicação web desenvolvida com **HTML, CSS e JavaScript** que realiza conversões de moedas em tempo real utilizando uma API externa.

---

## 🚀 Sobre o projeto

O Conversor de Moedas permite ao usuário converter valores entre diferentes moedas de forma rápida e precisa, consumindo dados atualizados diretamente de uma API REST.

O projeto foi desenvolvido com foco em:

* Prática de consumo de APIs
* Manipulação do DOM
* Validação de formulários
* Organização de código no front-end

* Exibição dinâmica de resultados
* Feedback visual para erros

---

## 🌐 Integração com API

O projeto consome a API ExchangeRate-API para obter taxas de câmbio atualizadas.

As requisições são feitas via **HTTP (GET)** e os dados são tratados em formato **JSON**, seguindo o padrão de APIs REST.
